练习

3.14. 练习#

Solutions to most exercises are available. We are happy to add or correct solutions. Please make contributions through GitHub.

Exercise: list expressions [★]

  • 构造一个包含整数 1 到 5 的列表。使用正方形 列表的括号符号。

  • 构造相同的列表，但不使用方括号表示法。相反， 使用 :: 和 []。

  • 再次构建相同的列表。这次一定会出现下面的表达式 在你的回答中：[2; 3; 4]。使用 @ 运算符，不要使用 ::。

Exercise: product [★★]

编写一个函数 product 返回 a 中所有元素的乘积 列表。空列表的所有元素的乘积是 1。

Exercise: concat [★★]

编写一个连接列表中所有字符串的函数。连接 空列表中所有字符串的总和是空字符串 ""。

Exercise: product test [★★]

对你在上面练习中编写的函数 product 进行单元测试。

Exercise: patterns [★★★]

使用模式匹配，编写三个函数，每个函数对应以下各项 属性。如果输入列表具有以下内容，你的函数应返回 true 财产，否则 false 。

  • 列表的第一个元素是 "bigred"

  • 该列表恰好有两个或四个元素；不要使用 length 函数

  • 列表的前两个元素相等

Exercise: library [★★★]

请查阅 List standard library 来解决这些练习：

  • 编写一个函数，它接受 int list 并返回第五个元素 该列表（如果存在这样的元素）。如果列表少于五个 元素，返回 0。 提示：List.length 和 List.nth。

  • 编写一个函数，它接受 int list 并返回排序的列表 降序排列。 提示：List.sort，其中 Stdlib.compare 作为第一个 参数，以及 List.rev.

Exercise: library test [★★★]

为你在中编写的每个函数编写几个 OUnit 单元测试 之前的练习。

Exercise: library puzzle [★★★]

  • 编写一个返回列表最后一个元素的函数。你的函数可能 假设列表非空。 提示：使用两个库函数，并执行 不要编写你自己的任何模式匹配代码。

  • 编写一个函数 any_zeros : int list -> bool 返回 true 如果并且 仅当输入列表包含至少一个 0 时。 提示：使用一个库 函数，并且不要编写你自己的任何模式匹配代码。

你的解决方案将只有一两行代码。

Exercise: take drop [★★★]

  • 编写一个函数 take : int -> 'a list -> 'a list 使得 take n lst 返回 lst 的前 n 元素。如果 lst 的元素少于 n， 返回全部。

  • 编写一个函数 drop : int -> 'a list -> 'a list 使得 drop n lst 返回 lst 的除第一个 n 元素之外的所有元素。如果 lst 少于 n 元素，返回空列表。

Exercise: take drop tail [★★★★]

如果不是，请将 take 和 drop 的解决方案修改为尾递归 已经。在具有较大 n 值的长列表上测试它们，看看它们是否 堆栈空间不足。要构造长列表，请使用 -- 运算符 lists 部分。

Exercise: unimodal [★★★]

编写一个函数 is_unimodal : int list -> bool ，它接受一个整数列表并 返回该列表是否是单峰的。 单峰列表是一个列表 单调增加到某个最大值，然后单调减少 那个值。其中一个或两个段（增加或减少）可以为空。一个 常量列表是单峰的，空列表也是如此。

Exercise: powerset [★★★]

编写一个函数 powerset : int list -> int list list ，它接受一组 S 表示为列表并返回 S 的所有子集的集合。顺序 幂集中的子集和子集中元素的顺序并不重要。

*提示：*考虑这个问题的递归结构。假设你已经 有p，这样p = powerset s。你如何使用 p 来计算 powerset (x :: s)？

Exercise: print int list rec [★★]

编写一个函数 print_int_list : int list -> unit 打印其输入列表， 每行一个数字。例如，print_int_list [1; 2; 3] 应该导致 这个输出：

1
2
3

下面是一些可以帮助你入门的代码：

let rec print_int_list = function
| [] -> ()
| h :: t -> (* fill in here *); print_int_list t

Exercise: print int list iter [★★]

编写一个函数 print_int_list' : int list -> unit ，其规范为 与 print_int_list 相同。不要在解决方案中使用关键字 rec，但是 而是使用 List module 函数 List.iter。这是一些代码 让你开始：

let print_int_list' lst =
  List.iter (fun x -> (* fill in here *)) lst

Exercise: student [★★]

假设以下类型定义：

type student = {first_name : string; last_name : string; gpa : float}

给出具有以下类型的 OCaml 表达式：

  • student

  • student -> string * string（提取学生姓名的函数）

  • string -> string -> float -> student （创建学生的函数 记录）

Exercise: pokerecord [★★]

这是一个代表一些 Pokémon 类型的变体：

type poketype = Normal | Fire | Water

  • 将类型 pokemon 定义为具有字段 name （字符串）、hp 的记录 （一个整数）和 ptype （一个 poketype）。

  • 创建一条名为 charizard 且类型为 pokemon 的记录，表示 Pokémon 拥有 78 HP 和火属性。

  • 创建一条名为 squirtle 且类型为 pokemon 的记录，表示 Pokémon 具有 44 HP 和水属性。

Exercise: safe hd and tl [★★]

编写一个函数 safe_hd : 'a list -> 'a option ，如果满足则返回 Some x 输入列表的头部是 x，如果输入列表为空，则为 None。

还编写一个函数 safe_tl : 'a list -> 'a list option 返回 列表的尾部，如果列表为空，则为 None 。

Exercise: pokefun [★★★]

编写一个函数 max_hp : pokemon list -> pokemon option ，给定一个列表 pokemon，找到HP最高的神奇é兽。

Exercise: date before [★★]

将“类似日期的三元组”定义为 int * int * int 类型的值。例子 类似日期的三元组包括 (2013, 2, 1) 和 (0, 0, 1000)。 日期是 类似日期的三元组，其第一部分是正年份（即共同年份中的年份） era)，第二部分是 1 到 12 之间的月份，第三部分是 1 之间的一天 和 31（或 30、29 或 28，具体取决于月份和年份）。 (2013, 2, 1) 是一个 日期； (0, 0, 1000) 不是。

编写一个函数 is_before ，将两个日期作为输入并计算结果 true 或 false。如果第一个参数是一个日期，则其计算结果为 true 出现在第二个参数之前。 （如果两个日期相同，则结果为 false。）

你的函数需要仅针对日期正确工作，而不是任意日期 类似日期的三元组。但是，你可能会发现更容易编写你的 如果你考虑使其适用于任意类似日期的三元组，则可以解决此问题。对于 例如，更容易忘记输入是否确实是日期，并且 只需编写一个函数来声明（例如）2013 年 1 月 100 日到来 2013 年 2 月 34 日之前—因为一月的任何日期都在任何日期之前 2 月，但有一个函数表示 2013 年 1 月 100 日晚于 2013 年 2 月 34 日同样有效。你可能会忽略闰年。

Exercise: earliest date [★★★]

编写一个函数earliest : (int*int*int) list -> (int * int * int) option。它 如果输入列表为空，则计算结果为 None；如果日期 d 为 Some d，则计算结果为 Some d 列表中最早的日期。 提示：使用is_before。

与上一个练习一样，你的函数需要正确运行 仅适用于日期，不适用于任意类似日期的三元组。

Exercise: assoc list [★]

使用函数 insert 和 lookup 关联列表一节 构造一个关联列表 将整数 1 映射到字符串“一”，将 2 映射到“二”，将 3 映射到“三”。查找 键 2. 查找键 4.

Exercise: cards [★★]

  • 定义代表四种花色的变体类型 suit ♣ ♦ ♥ ♠，在 standard 52-card deck 中。所有构造函数 你的类型应该是恒定的。

  • 定义一个类型 rank 代表一张牌可能的等级：2, 3, …, 10、杰克、王后、国王或王牌。有很多可能的解决方案；你自由了 选择适合你的任何内容。一是让rank成为int的同义词， 并假设 Jack=11，Queen=12，King=13，Ace=1 或 14。另一种方法是 使用变体。

  • 定义一个类型 card 来表示单张牌的花色和点数。制造 它有两个字段的记录。

  • 定义一些 card 类型的值：梅花 A、红心 Q、 方块二，黑桃七。

Exercise: matching [★]

对于下面列表中的每个模式，给出 int option list 类型的值 确实不匹配模式并且不是空列表，或者解释为什么会这样 不可能的。

  • Some x :: tl

  • [Some 3110; None]

  • [Some x; _]

  • h1 :: h2 :: tl

  • h :: tl

Exercise: quadrant [★★]

完成下面的 quadrant 函数，该函数应返回象限 根据右图给出 x, y 点（借自 Wikipedia)。 位于轴上的点不属于任何象限。 提示：(a) 定义 用于整数符号的辅助函数，(b) 与一对匹配。

type quad = I | II | III | IV
type sign = Neg | Zero | Pos

let sign (x:int) : sign =
  ...

let quadrant : int*int -> quad option = fun (x,y) ->
  match ... with
    | ... -> Some I
    | ... -> Some II
    | ... -> Some III
    | ... -> Some IV
    | ... -> None

Exercise: quadrant when [★★]

重写象限函数以使用 when 语法。你将不需要你的 之前的辅助函数。

let quadrant_when : int*int -> quad option = function
    | ... when ... -> Some I
    | ... when ... -> Some II
    | ... when ... -> Some III
    | ... when ... -> Some IV
    | ... -> None

Exercise: depth [★★]

编写一个函数 depth : 'a tree -> int 返回节点数 从根到叶子的任何最长路径。例如，空的深度 树（简单来说Leaf）是0，上面的树t的深度是3。 提示： 有一个库函数 max : 'a -> 'a -> 'a 返回最大值 任意两个相同类型的值。

Exercise: shape [★★★]

编写一个函数 same_shape : 'a tree -> 'b tree -> bool 来确定 两棵树是否具有相同的形状，无论它们的值是否相同 每个节点的进位都是相同的。 提示：使用具有三个分支的模式匹配， 其中匹配的表达式是一对树。

Exercise: list max exn [★★]

编写一个返回最大整数的函数 list_max : int list -> int 在列表中，或者如果列表为空则引发 Failure "empty" 。

Exercise: list max exn string [★★]

编写一个返回字符串的函数 list_max_string : int list -> string 包含列表中的最大整数，或字符串 "empty" （注意，不是 如果列表为空，则抛出异常 Failure "empty" 但只是字符串 "empty")。 提示：标准库中的 string_of_int 将执行其名称所暗示的操作。

Exercise: list max exn ounit [★]

编写两个 OUnit 测试来确定你的解决方案是否列出 max exn， 上面，当其输入为空列表时正确引发异常，并且 当该列表为时，它是否正确返回输入列表的最大值 非空。

Exercise: is_bst [★★★★]

编写一个函数 is_bst : ('a*'b) tree -> bool ，当且仅返回 true 如果给定的树满足二叉搜索树不变量。一个高效的 该函数最多访问每个节点一次的版本有点棘手 写。 提示：编写一个递归辅助函数，该函数采用树和 给你 (i) 树中的最小值和最大值，或者 (ii) 告诉你 树是空的，或者 (iii) 告诉你树不满足 不变的。你的 is_bst 函数不会递归，但会调用你的 辅助函数和模式匹配结果。你将需要定义一个新的 辅助函数的返回类型的变体类型。

Exercise: quadrant poly [★★]

修改象限的定义以使用多态变体。 你的函数类型应该变成这些：

val sign : int -> [> `Neg | `Pos | `Zero ]
val quadrant : int * int -> [> `I | `II | `III | `IV ] option
