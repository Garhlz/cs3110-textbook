# 数据和类型

在本章中，我们将研究 OCaml 的一些内置数据类型，包括列表、变体、记录、元组和选项。其中许多可能会让你觉得很熟悉，因为类似的类型在其他编程语言中也很常见。特别是：

- **列表**和**元组**可能与 Python 中的相似；
- **记录**和**变体**可能与 C 或 Java 中的 `struct` 和 `enum` 类型相似。

由于这种相似性，我们把这些称为*标准*数据类型。我们还会学习*模式匹配*，这个特性你可能不太熟悉。

几乎在学完列表之后，我们就会暂停对标准数据类型的学习，转而学习如何用 OUnit 在 OCaml 中进行单元测试。OUnit 是一个单元测试框架，类似于你可能在其他语言中用过的框架。我们之所以现在才介绍 OUnit，是因为它依赖于列表。

在本章后面，我们会研究一些在其他语言中不那么常见的 OCaml 数据类型。它们包括：

- **选项**，与 Java 中的 `null` 有松散的联系；
- **关联列表**，这是一个基于列表和元组的极简映射（字典）实现；
- **代数数据类型**，可以说是 OCaml 中最重要的类型，也是许多其他内置类型的基础；
- **异常**，这是一种特殊的代数数据类型。
